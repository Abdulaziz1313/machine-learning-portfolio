## Overview

This repository contains my machine learning portfolio. It includes a range of notebooks covering different machine learning topics such as classification, clustering, neural networks, reinforcement learning, and deep learning.

Most of the notebooks use a dataset about the impact of AI on student performance. The reinforcement learning notebook uses the FrozenLake environment from Gymnasium.

The purpose of this portfolio is to demonstrate practical work across different machine learning approaches rather than focusing on only one method.

## Summary of Work

### KNN
This notebook applies k-Nearest Neighbour to classify student performance categories based on academic and AI-related features.

### KMeans
This notebook uses k-Means clustering to group students without using labels directly and then compares the resulting clusters with known performance categories.

### SVM
This notebook uses Support Vector Machine classification and compares different kernels to find the best-performing version of the model.

### Neural Networks
This notebook uses an Artificial Neural Network with different hidden layer settings to classify performance categories.

### Reinforcement Learning
This notebook uses Q-learning with the FrozenLake environment to demonstrate reinforcement learning using states, actions, rewards, and a learned Q-table.

### Deep Learning
This notebook uses TensorFlow/Keras to build a deeper neural network model and compare different deep learning architectures.
